Case Study: Web app for managing Product Master Data for a Retail Client Business Problem: • Merchants were managing master data manually through Excel-based Redbooks resulting in inefficient product data management. Solution: • Automated data management and integration through a Web User Interface • Custom Web based solution was built to enable merchants effectively manage master data. • Automated workflows to trigger these data pipelines for seamless updates Tech Stack: • T-SQL, SQL Server, Dataverse, API Management, SSIS, ETL, Python, PowerApps, Power Automate, Power Pages Outcomes: • Improved data accuracy (by ~60%) and reduced operational costs (~40%)
